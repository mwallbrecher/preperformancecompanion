# Pre-Performance Companion

A Flutter-based UX prototype for an MSc User Experience Design project at Kingston University London.

The app explores short, in-the-moment support for users who feel nervous, overwhelmed, distracted, or stuck in overthinking shortly before a performance situation such as a presentation, interview, meeting, or social event.

## Concept

The prototype focuses on quick pre-performance grounding through four lightweight modes:

- **Breathing Exercise** – guided breathing with touch interaction
- **Balancing Game** – short sensor-based attention reset
- **Thought Offloading** – voice-based worry/thought capture
- **Focus Fill Mode** – passive visual focus timer

The project is not intended as a clinical or therapeutic mental health tool. It is a UX research prototype exploring perceived clarity, focus, and readiness in short pre-performance moments.

## Prototype Status

This is a mid-fidelity research prototype built for formative UX testing.  
It prioritises interaction logic, feature testing, and user feedback over production-level robustness.

## AI Use

AI tools were used as development support. Claude Code helped translate selected Figma components and design directions into Flutter widgets. ChatGPT was used for project organisation, wording refinement, and research structuring.

All product ideas, feature decisions, UX direction, research interpretation, and final implementation decisions were made by the project author. AI outputs were reviewed, adapted, and directed through follow-up prompts.
